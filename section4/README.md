> WL.BusyIndicator

	busyIndicator = new WL.BusyIndicator(null, {text : 'Loading...'});

- arg1: parent element ID, is ignored in iOS, Andaroid, Windows Phone, and BlackBerry environments.
- arg2: available options: text, color, fullScreen etc.
